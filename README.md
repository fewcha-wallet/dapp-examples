## Development

### Install

```
yarn
```

### Build

```
yarn build
```

### Publish

```
yarn build
yarn run publish
```

### Web3-react

```
yarn build
```

Keep a terminal for

```
yarn dev start
```

Change `packages/web3-react` and run `yarn build` again
